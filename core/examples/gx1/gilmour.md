# How to Use This Document

Each patch in this library is designed for the BOSS GX-1. The GX-1 allows you to reorder effects within the signal chain on a per-memory basis — a key feature for achieving historically accurate Gilmour tones.

Every patch includes a Recommended Signal Order callout (amber highlighted box). Follow this order when programming the patch. Where it differs from the GX-1 default, it will be marked REORDER REQUIRED.

To reorder effects on the GX-1: press [EDIT] → turn knob [1] to select the block to move → press knob [1] to make it movable → turn knob [1] to the new position → press knob [1] to confirm.

## General Gilmour Principles

Amp gain is almost always low — the overdrive/fuzz pedal provides the distortion, not the amp. Keeping the amp clean preserves note definition and dynamics.

MUFF FUZZ (Big Muff π) is the primary lead pedal from Meddle through The Wall. 60S FUZZ (Fuzz Face) is correct for DSOTM and Animals era material. T-SCREAM (TS-808) is used only as a boost/tightener, never as the main voice.

ANALOG delay is almost always the right choice. Warm, dark repeats sit behind the guitar rather than competing with it. MODULATE delay is used only for Marooned where wavering repeats suit the slide guitar.

SCRIPT PH (MXR Phase 90 model) is the correct phaser for all mid-70s material. Keep the rate slow — it should feel like the guitar is breathing, not like an effect is spinning.

Mic choices lean warm and dark across the library. Gilmour's tone is never bright or harsh. RIBON121 (Royer R-121) is the darkest choice for intimate clean tones; BLEND A (SM57 + Ribbon) is the default for leads.

# Comfortably Numb

The Wall (1979)

## [CN-1]  Verse Rhythm

Goal: The slow, hypnotic clean rhythm. Heavily chorused, almost dreamlike — it floats rather than drives. Gilmour played this through a clean Hiwatt with a strong chorus. One of the most recognisable rhythm tones in rock.

Recommended Signal Order:  FX1 (Chorus) → AMP/CAB → FX2 (Phaser) → NS → DELAY → REVERB

Note: OD/DS block is OFF and bypassed. Move Phaser to FX2 so it sits after the amp, colouring the clean tone without adding fuzz interaction.

| Block | Setting | Why |
| --- | --- | --- |
| AMP/CAB | BRIT STACK, GAIN: 18, BASS: 50, MID: 48, TREB: 58, SP: 4×12", MIC: RIBON121 | Very low gain — clean platform. Ribbon mic for warmth. The chorus does all the work. |
| FX1 Chorus | STEREO, RATE: 28, DEPTH: 55, LEVEL: 70, PRE-DELAY: 4ms | Centrepiece of the tone. Deeper and more prominent than any other chorus patch — thick and lush, almost like two guitars. |
| FX2 Phaser | SCRIPT PH, RATE: 15, DEPTH: 40, LEVEL: 55 | Slow phase underneath the chorus adds a psychedelic, slightly woozy quality. Keep subtle — the chorus is the star. |

Goal: The main clean body of the song — chorus-drenched, slightly liquid. Earlier than the Wall era so slightly rawer and less polished.

Recommended Signal Order:  FX1 (Chorus) → AMP/CAB → FX2 (Phaser) → NS → DELAY → REVERB

Note: OD/DS is OFF. Chorus in FX1 before the amp gives a slightly more saturated, embedded modulation quality — characteristic of early 70s Gilmour recordings.

## [ECH-2]  The "Seagull" / Chaos Section

Goal: The infamous middle section — alien, wailing seagull sound. Gilmour used a wah in a fixed position with the pickup selector between positions creating feedback. Approximated here with fixed flanger and phaser.

Recommended Signal Order:  FX1 (Flanger) → OD/DS (Lead DS) → AMP/CAB → FX2 (Phaser) → DELAY → REVERB

Note: NS is intentionally OFF — the only patch in the library where this is the case. High-resonance Flanger before the high-gain amp creates the seagull character. Leave NS off to allow feedback and noise.

| Block | Setting | Why |
| --- | --- | --- |
| AMP/CAB | BRIT STACK, GAIN: 75, BASS: 45, MID: 65, TREB: 70, SP: 4×12", MIC: DYN57 | High gain for feedback. Mids pushed high to encourage harmonic feedback at lower volumes. |
| FX1 Flanger | RATE: 8, DEPTH: 80, MANUAL: 55, RESO: 70, LEVEL: 75 | High resonance creates that swooping, jet-like character. Low rate so it moves slowly and dramatically. |
| FX2 Phaser | 12 STAGE, RATE: 10, DEPTH: 80, RESO: 65, LEVEL: 70 | 12-stage phaser at very slow rate adds a second layer of swirling movement beneath the flanger. |
| OD/DS | LEAD DS, DRIVE: 80, TONE: +5, LEVEL: 65 | High gain distortion for feedback sustain — LEAD DS has a smooth top end that doesn't become piercing. |
| DELAY | ANALOG, TIME: 680ms, FEEDBACK: 55, LEVEL: 65, HIGH CUT: 4kHz | Long delay with high feedback — approaching self-oscillation. Intentional for the chaos section. |
| REVERB | HALL S, TIME: 4.5s, TONE: +5, PRE-DELAY: 10ms, LEVEL: 55, DENSITY: 8 | Bright, dense, cavernous — should feel like filling an enormous space. |
| NS | OFF | Intentionally off — you want the noise and feedback for this section. |

# Marooned

The Division Bell (1994)

## [MAR-1]  Slide Rhythm / Atmosphere

Goal: The ambient, hovering background tone. Slow, steel-like, spacious. This song is almost entirely about texture and space. The MODULATE delay and SHIMMER reverb are key.

Recommended Signal Order:  FX1 (Chorus) → AMP/CAB → FX2 (Tremolo) → NS → MODULATE DELAY → SHIMMER REVERB

Note: OD/DS is OFF. MODULATE delay and SHIMMER reverb are the defining elements — wavering repeats and ascending pitch tail. FX3 is available for additional texture if needed.

| Block | Setting | Why |
| --- | --- | --- |
| AMP/CAB | TWIN, GAIN: 16, BASS: 50, MID: 52, TREB: 60, SP: ORIGINAL, MIC: CND87 | By the 90s Gilmour's clean tones were more hi-fi. CND87 gives a flatter, more open response. |
| FX1 Chorus | STEREO, RATE: 14, DEPTH: 38, LEVEL: 60, PRE-DELAY: 4ms | Slow, gentle chorus — adds the steel-guitar shimmer to slide notes. |
| FX2 Tremolo | RATE: 22, DEPTH: 30, LEVEL: 65 | Very slow, subtle tremolo — barely perceptible but gives notes a gentle pulse. |
| OD/DS | OFF | — |
| DELAY | MODULATE, TIME: 580ms, FEEDBACK: 38, LEVEL: 50, HIGH CUT: 2.5kHz, MOD RATE: 12, MOD DEPTH: 18 | MODULATE delay adds a gentle wavering to the repeats — gives the atmosphere a slightly liquid, unstable quality. |
| REVERB | SHIMMER, TIME: 4.0s, TONE: -5, PRE-DELAY: 30ms, LEVEL: 48, PITCH: +12, PITCH LVL: 25 | SHIMMER reverb with subtle +12 pitch shift — a very 90s production sound, perfect for this track. |
| NS | THRESHOLD: 22, RELEASE: 65 | Very low threshold, very long release — let everything breathe and sustain. |

## [MAR-2]  Main Slide Lead

Goal: The central slide melody — one of Gilmour's most emotionally devastating performances. The tone is smoother and more controlled than his earlier leads. The sustain comes almost entirely from compression and a very restrained Muff.

Recommended Signal Order:  FX1 (Compressor) → OD/DS (Muff Fuzz) → AMP/CAB → FX2 (Chorus) → NS → MODULATE DELAY → SHIMMER REVERB

Note: Compressor before Muff is correct here — the Muff drive is very low (45) so the compressor adds sustain rather than killing dynamics. SHIMMER reverb PITCH LVL is slightly higher than MAR-1.

| Block | Setting | Why |
| --- | --- | --- |
| AMP/CAB | BRIT STACK, GAIN: 25, BASS: 53, MID: 58, TREB: 60, SP: 4×12", MIC: BLEND A | Lower gain than most leads — 90s Gilmour tone is cleaner. The sustain comes from compression and the Muff. |
| FX1 Compressor | D-COMP, SUSTAIN: 80, ATTACK: 35, LEVEL: 65 | Second highest sustain in the library — slide notes need even more sustain than fretted notes to sing properly. |
| OD/DS | MUFF FUZZ, DRIVE: 45, TONE: +3, LEVEL: 65 | Lowest Muff drive in the library — by this era Gilmour used the Big Muff as a subtle, transparent sustain tool. |
| FX2 Chorus | STEREO, RATE: 16, DEPTH: 30, LEVEL: 55, PRE-DELAY: 3ms | Carries the shimmer quality from the rhythm patch into the lead. |
| DELAY | MODULATE, TIME: 520ms, FEEDBACK: 40, LEVEL: 55, HIGH CUT: 3.15kHz, MOD RATE: 10, MOD DEPTH: 15 | MODULATE delay again — wavering repeats suit slide guitar beautifully, making notes feel like they're drifting. |
| REVERB | SHIMMER, TIME: 4.5s, TONE: -3, PRE-DELAY: 20ms, LEVEL: 45, PITCH: +12, PITCH LVL: 30 | Shimmer reverb with slightly more PITCH LVL than the rhythm patch — lead notes should feel like they're ascending. |
| NS | THRESHOLD: 28, RELEASE: 60 | Long release — slide notes decay slowly and the gate must never cut them short. |

# Effect Choices — Reference Guide

This table explains why specific effects and models were chosen across the library.

| Effect / Model | Why it was chosen |
| --- | --- |
| MUFF FUZZ | Big Muff π. Gilmour's primary lead pedal from Meddle (1971) through The Wall (1979). Medium drive — never maxed out. |
| 60S FUZZ | Fuzz Face. Used more on DSOTM and Animals. More dynamic and touch-sensitive than the Muff — correct for the 1971–77 era. |
| T-SCREAM | Ibanez TS-808. Used only as a boost/tightener into the amp on rhythm patches, never as the main distortion voice. |
| SCRIPT PH | MXR Phase 90 (script logo version). The correct phaser for all mid-70s Gilmour material. Slow rate always. |
| ANALOG delay | Gilmour almost exclusively used analog-voiced delays. Warm, dark repeats that sit behind the guitar. |
| MODULATE delay | Used only for Marooned (1994). The gentle wavering of the repeats suits slide guitar and the ambient 90s production style. |
| BRIT STACK | Models a Marshall 1959 — the closest available model to Gilmour's Hiwatt DR103. Used for all lead tones. |
| TWIN | Fender Twin Reverb. Used for clean rhythm tones on WYWH/SOYCD era — maximum clean headroom. |
| DELUXE | Fender Deluxe Reverb. Used for intimate, warm clean tones (Breathe rhythm, Hey You rhythm) where the Twin would be too big. |
| SHIMMER reverb | Used exclusively for Marooned. The +12 pitch shift in the reverb tail is a defining sound of 90s Pink Floyd production. |
| BLEND A mic | SM57 + Royer R-121 (SM57 louder). Default for lead tones — present but warm. No harshness. |
| RIBON121 mic | Royer R-121 ribbon mic. Darkest, warmest choice. Used for intimate clean tones where any harshness would be wrong. |
| D-COMP | MXR Dyna Comp. More aggressive sustain — used for all lead patches requiring long, singing note decay. |
| ORANGE comp | Dan Armstrong Orange Squeezer. Gentlest compressor — used for fingerpicked clean tones where touch sensitivity must be preserved. |
